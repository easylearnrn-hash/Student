# Make student-notes Bucket Public

**Quick Fix - Do this in Supabase Dashboard:**

1. Go to **Storage** in Supabase Dashboard
2. Click on **student-notes** bucket
3. Click the **Settings** icon (gear icon)
4. Toggle **Public bucket** to ON
5. Click **Save**

This will make all files accessible via public URLs without needing authentication or signed URLs.

After doing this, refresh the student portal and the PDFs should load!
