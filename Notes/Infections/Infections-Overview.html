Infections & Isolation Overview

Transmission-based precautions, PPE selection, and disease isolation reference

Standard Precautions

Applied to ALL patients regardless of diagnosis — blood, body fluids, non-intact skin, mucous membranes.

  • Hand hygiene (before and after patient contact)
  • Gloves when touching blood, fluids, secretions
  • Gown if splashing likely
  • Mask/eye protection if aerosol risk
  • Sharps safety, cough etiquette

Diseases using standard only: HIV Hep B Hep C CMV Aspergillus

PPE: Donning & Doffing Order

🟢 DONNING ORDER (Putting On)
  1. Hand hygiene
  2. Gown
  3. Mask / N95
  4. Goggles / Face shield
  5. Gloves (over gown cuffs)
🔴 DOFFING ORDER (Taking Off)
  1. Gloves (most contaminated)
  2. Goggles / Face shield
  3. Gown
  4. Mask / N95 (least contaminated)
  5. Hand hygiene
📏 Droplet Distance Rule

Surgical mask required within 3 feet (CDC) or 6 feet (some standards). Particles >5 µm fall quickly.

🌀 Airborne vs. Droplet

Airborne: particles <5 µm, stay suspended — N95 + negative-pressure room required

Droplet: particles >5 µm, fall quickly — surgical mask sufficient

NCLEX Must-Know

  • Doff gloves FIRST (most contaminated), mask LAST
  • N95 = always for TB, Measles, Chickenpox, SARS/COVID aerosol-generating procedures
  • Surgical mask = sufficient for droplet-only diseases (flu, mumps, rubella)
  • C. diff → soap + water ONLY (no alcohol gel)
  • Negative-pressure room = airborne diseases
  • Chickenpox requires BOTH airborne + contact (dual precautions)
🧠
NCLEX KEY:

"Which PPE is required?" — Identify the transmission type first, then select accordingly. N95 is ONLY for airborne (TB/measles/varicella); surgical mask for droplet diseases.
