Blood Vessels & Lymphatic System

Blood composition, vessel types, WBC differential, lymph nodes, and clotting

Blood Composition

Blood = Plasma 55% + Formed Elements 45%

Plasma 55%
RBCs 44%
  • Plasma (55%): Water (91%), proteins (albumin, globulins, fibrinogen), nutrients, hormones, gases, waste
  • RBCs (44%): Erythrocytes — carry O₂ via hemoglobin, no nucleus, 120-day lifespan
  • WBCs (<1%): Leukocytes — immune defense; 5 types
  • Platelets: Thrombocytes — clotting fragments from megakaryocytes; 150,000–400,000/μL

Clotting Process

Hemostasis Steps

  • 1. Vascular Spasm: Vessel constricts immediately
  • 2. Platelet Plug: Platelets aggregate at injury site
  • 3. Coagulation Cascade: Clotting factors activate → fibrin mesh forms
  • 4. Clot Retraction: Clot tightens
  • 5. Fibrinolysis: Plasmin breaks down clot when healed

Key Clotting Labs

  • Platelets: 150,000–400,000/μL
  • PT (Prothrombin Time): 10–13 sec — extrinsic pathway (monitors Warfarin)
  • aPTT: 25–35 sec — intrinsic pathway (monitors Heparin)
  • INR: ~1 (normal); 2–3 (therapeutic anticoagulation for Warfarin)
  • Fibrinogen: 200–400 mg/dL
Memory: PT/INR = Warfarin (W for Watchful eye on Warfarin). aPTT = Heparin (H for Heparin). Extrinsic pathway = PT. Intrinsic pathway = aPTT.
Lymphatic System

Function

  • Returns excess interstitial fluid to venous circulation
  • Transports dietary fats (chylomicrons) from GI tract
  • Filters pathogens via lymph nodes
  • Produces lymphocytes (immune defense)
  • Lymph drains into thoracic duct → left subclavian vein

Lymph Nodes

  • Bean-shaped structures that filter lymph
  • Key locations: cervical, axillary, inguinal, mediastinal, mesenteric
  • Lymphadenopathy: enlarged lymph nodes → infection, cancer
  • Spleen: largest lymphoid organ — filters blood, stores RBCs/platelets
  • Thymus: T-cell maturation (prominent in children)
Clinical Note: Edema can result from lymphatic obstruction (e.g., after mastectomy → lymphedema of arm), low albumin (reduces oncotic pressure), or increased capillary pressure (HF, venous insufficiency).
Normal Blood Values Reference

RBC / Hemoglobin

  • RBC: M: 4.5–5.5, F: 4–5 (×10⁶/μL)
  • Hgb: M: 13.5–17.5, F: 12–16 g/dL
  • Hct: M: 40–52%, F: 37–47%

WBC

  • Total WBC: 5,000–10,000/μL
  • <1,000 neutrophils = neutropenia (infection risk)
  • >11,000 = leukocytosis (infection/inflammation)

Platelets

  • Normal: 150,000–400,000/μL
  • <50,000 = bleeding risk
  • <20,000 = spontaneous bleeding
  • >1,000,000 = thrombocytosis
