Insulin Storage & Blood Glucose Checking

Refrigeration rules, 28-day expiration after opening, vial handling, fingerstick glucose steps

Insulin Storage Guidelines

1. Refrigeration (Unopened Vials)

  • Store all unopened vials in the refrigerator (2–8°C / 36–46°F)
  • Organize fridge: open vials on one side, unopened vials on the other
  • Keep away from freezer compartment — never freeze insulin
  • Never store in a car — temperature extremes degrade insulin

2. Before Opening — Check & Label

  • Check the expiration date on the vial
  • If expired → discard immediately
  • If valid → label the vial with the date opened

3. After Opening — 28-Day Rule

  • New expiration = 28–30 days from the opening date
  • Store opened vials in the refrigerator OR at room temperature
  • Keep away from direct sunlight and heat sources
  • Room temperature storage is fine for up to 28 days (improves comfort with injections)
Key Rule: Opened insulin expires in 28 days regardless of where it's stored. Always label with the date opened. If unlabeled → discard.
Preparing Insulin for Administration

Warming & Handling

  • If refrigerated: roll vial in palms to warm before injection — prevents discomfort
  • NEVER shake insulin — shaking creates bubbles and denatures the protein
  • Cloudy (NPH): roll gently between palms to mix and warm
  • Clear (Regular, Rapid, Long): do not roll — ready to use as-is

If Accidentally Shaken

  • Allow vial to rest (sit undisturbed) before use
  • Inspect for visible particles or clumping before drawing
  • If appearance is abnormal → discard

Insulin Patch Pumps

Patch Pump Storage Note

  • Insulin patch pumps contain rapid-acting insulin only
  • Infusion sets should be changed every 2–3 days (48–72 hours)
  • Leaving set >4 days → infection, lipodystrophy, absorption issues
Steps to Check Blood Glucose (Fingerstick)

8-Step Protocol

  1. Insert the test strip into the glucose meter
  2. Clean the fingertip with an alcohol swab; allow to dry
  3. Prick the finger with a lancet to draw blood (use side of fingertip, not tip center)
  4. Gently squeeze the finger to encourage a drop to form
  5. Wipe away the first drop of blood (first drop may contain tissue fluid)
  6. Place the second drop of blood on the test strip in the meter
  7. Apply an alcohol swab or bandage to the fingertip to stop bleeding
  8. Read results from the glucose meter and document
Why wipe away the first drop? It may be diluted with tissue fluid (interstitial fluid), giving a falsely low reading. Always use the second drop for accuracy.
NCLEX Must-Know

Top NCLEX Points

  • Opened insulin expires in 28 days — label with date opened
  • Unopened insulin → refrigerator until use
  • NEVER freeze, NEVER shake insulin
  • Cloudy Regular insulin → discard (should always be clear)
  • NPH cloudy → normal; roll gently
  • Warm refrigerated insulin in palms before injecting
  • Fingerstick: wipe away first drop, use second drop
  • Rotate injection sites to prevent lipodystrophy
  • Patch pump: change infusion set every 48–72 hours
